Moxy — Character Test

All drawn/overridden glyphs at 7 weights, using the installed Moxy variable font.

two-master (thickens with weight) single-master (constant across weight)

Uses the locally-installed Moxy VF (font-variation-settings: "wght" N). If glyphs render as fallback boxes, run make install-vf and reload.

What to look for: two-master glyphs (green) should visibly thicken from 300→900. Single-master glyphs (amber) stay constant — that's expected for $ @ & (they replaced constant-weight grafts), but a bug for numerals that sit alongside two-master ones (2 and 5 should thicken like 4 and 7).
